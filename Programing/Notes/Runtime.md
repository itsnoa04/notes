Runtime refers to the time it takes to execute a program or the time period in which the program is running.

Runtime changes as device changes as it depends on CPU, Programing Language ,etc .

This makes Runtime a bad way to measure efficency of code.
Therefore we use [[Big O Asymptotic Analysis]].
