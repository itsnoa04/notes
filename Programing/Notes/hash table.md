Hashtables or object is one of the [[DataStructures]] which has a key and a value.
```js
let user = {
	age:18,
	magic:true,
	scream: function(){
		console.log('ahh!')
	}
}
```


Keys are generated by the use of a [[Hash Function]].

most [[Hash methods]] are [[O(1)]].
